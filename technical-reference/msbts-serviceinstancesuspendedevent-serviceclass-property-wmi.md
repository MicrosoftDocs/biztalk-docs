# MSBTS\_ServiceInstanceSuspendedEvent.ServiceClass Property (WMI)

Â 

Contains the class of the service instance to which this message belongs. The syntax shown is language neutral.

## Syntax

```C#
  
uint32 ServiceClass;  
```

## Remarks

This property is read-only.
