# Configuration Wizard, Windows SharePoint Services Page

 

Use the **Windows SharePoint Services** page to configure the Windows SharePoint Services adapter on this computer. The BizTalk Server adapter for Windows SharePoint Services provides a tighter integration with Windows SharePoint Services and Microsoft Office InfoPath.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Enable the Windows SharePoint Services Adapter on this computer</strong></td>
<td>Select <strong>Enable the Windows SharePoint Services Adapter on this computer</strong> to enable the adapter on this computer.</td>
</tr>
<tr class="even">
<td><strong>Windows group</strong></td>
<td>The Windows group list provides an editable view of the BizTalk SharePoint Adapter Enabled Hosts Windows group.</td>
</tr>
<tr class="odd">
<td><strong>Windows SharePoint Services Adapter Web site</strong></td>
<td>Select the Web site that will host the Windows SharePoint Services adapter Web service.</td>
</tr>
</tbody>
</table>
