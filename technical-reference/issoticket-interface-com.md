# ISSOTicket Interface (COM)

 

The **ISSOTicket** interface issues and redeems Enterprise Single Sign-On (SSO) server tickets.

## Requirements

**Type Library:** SSOLookup 1.0 Type Library (SSOLookup.dll)

**Platforms:**  Windows
