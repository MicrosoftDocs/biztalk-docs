# MSBTS\_MessageInstance.ServiceClass Property (WMI)

Â 

Contains the name of the service class that corresponds to the message instance.

*The syntax shown is language neutral.*

## Syntax

```C#
  
uint32 ServiceClass;  
```

## Remarks

This property is read-only.
