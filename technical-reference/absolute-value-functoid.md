# Absolute Value Functoid

 

Use the **Absolute Value** functoid ( ![](images/Aa561898.f16947e6-6c6e-4a38-bc1f-446d8e0d2e17(BTS.80).jpeg)) to calculate the absolute value of the specified value.

## Input

**Parameter 1:** A numeric value, positive or negative, for which the absolute value is sought.

## Output

**Output 1:** A numeric value that is the absolute value of the specified numeric value.
