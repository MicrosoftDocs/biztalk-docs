# Method (Grid Property)

Â 

Use the **Method** property to specify the overall method for producing the output of the transformation.

## Category

Custom Header

## Allowed Values

<table>
<thead>
<tr class="header">
<th>Drop-down list choice</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>xml</strong></td>
<td>Produce the output of the transformation as XML.</td>
</tr>
<tr class="even">
<td><strong>html</strong></td>
<td>Produce the output of the transformation as HTML.</td>
</tr>
<tr class="odd">
<td><strong>text</strong></td>
<td>Produce the output of the transformation as text.</td>
</tr>
</tbody>
</table>


## Default Value

**xml**

## Remarks

This property provides a value for the **method** attribute of the XSL **output** element. For more information about this attribute, refer to the .NET Framework documentation.


> [!NOTE]
> <P>You cannot undo or redo the <STRONG>Method</STRONG> property.</P>
