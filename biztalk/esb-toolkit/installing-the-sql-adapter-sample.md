# Installing the SQL Adapter Sample
This section describes the process for installing the SQL Adapter sample. You must install the samples in [Installing and Running the Itinerary On-Ramp Sample](../esb-toolkit/installing-and-running-the-itinerary-on-ramp-sample.md) and [Installing and Running the Dynamic Resolution Sample](../esb-toolkit/installing-and-running-the-dynamic-resolution-sample.md) before you install and use this sample.  
  
 **To install the SQL Adapter sample**  
  
1.  In Windows Explorer, open the folder \Source\Samples\SqlAdapter\Install\Scripts.  
  
2.  Double-click the script file named Setup_bin.cmd.
