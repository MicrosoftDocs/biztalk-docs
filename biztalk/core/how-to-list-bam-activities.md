# How to List BAM Activities
Administrators use the **get-activities** command of the BAM Management utility to list the activities in the local BAM Primary Import database.  
  
### To list BAM Activities  
  
1.  Open a command prompt as follows: Click **Start**, click **Run**, type **cmd**, and then click **OK**.  
  
2.  Navigate to the tracking folder by typing **C:\Program Files\Microsoft BizTalk Server \<version>\Tracking** at the command prompt. Press **ENTER**.  
  
3.  Type **bm get-activities**.  
  
4.  Press **ENTER**.
