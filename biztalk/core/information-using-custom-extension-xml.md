# Information - Using Custom Extension XML
**Error Code**  
  
 btm1074  
  
 **Explanation**  
  
 BizTalk Mapper has generated its own Extension XML and appended it to the Custom Extension XML specified by the user.  
  
 **User Action**  
  
 Informational. No action required.
