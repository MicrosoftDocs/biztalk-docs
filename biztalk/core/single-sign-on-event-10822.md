# Single Sign-On: Event 10822
## Details  
  
| Field | Error Details |
|-----------------|------------------------------------------------------------|
|  Product Name   |                 Enterprise Single Sign-On                  |

|    Component    |                            N/A                             |
|  Symbolic Name  |                    ENTSSO_E_NO_ADAPTER                     |
|  Message Text   |           The specified adapter does not exist.            |
