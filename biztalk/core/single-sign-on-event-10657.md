# Single Sign-On: Event 10657

|  Event Source   |                           ENTSSO                           |
|    Component    |                            N\A                             |
|  Symbolic Name  |          SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTED           |
|  Message Text   |     Password sync for external adapters has started.%r     |
