# Error- A Nil Value functoid is connected to a target node &#39;{0}&#39; which has child link(s).
**Error Code**  
  
 btm1081  
  
 **User Action**  
  
 Either delete the links to the child nodes of the target node or remove the link to the nil value functoid.
