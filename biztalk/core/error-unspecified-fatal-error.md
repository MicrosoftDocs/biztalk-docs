# Error - Unspecified Fatal Error
**Error Code**  
  
 BEC1001, BEC2004, BEC3001  
  
 **Explanation**  
  
 An unspecified fatal error has occurred during schema compilation.  
  
 **User Action**  
  
 Make sure the schema file is valid and not corrupted.
