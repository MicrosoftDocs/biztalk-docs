| Field | Error Details |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
|  Product Name   |                                                             Enterprise Single Sign-On                                                              |

|  Message Text   | Could not access the SSO database. If this condition persists, the SSO service will go offline.%r<br /><br /> %1.%r<br /><br /> SQL Error code: %2 |
  
## Explanation

## User Action
