# Error - Cannot resolve schemaLocation attribute
**Error Code**  
  
 BEC8001  
  
 **Explanation**  
  
 BizTalk editor cannot find a schema imported by the schema you are opening.  
  
 **User Action**  
  
 Click the **Schema** node. In the Properties window, click the **Imports** property. The ellipsis (**â€¦**) button becomes active. Click the button. The **Imports** dialog appears. Edit the path to the imported schema in the dialog.
