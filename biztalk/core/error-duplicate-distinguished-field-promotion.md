# Error - Duplicate Distinguished Field Promotion
**Error Code**  
  
 BEC2015  
  
 **Explanation**  
  
 The indicated node in this schema is being promoted more than once as a Distinguished Field.  
  
 **User Action**  
  
 Use the **Distinguished Fields** tab of the **Promote Properties** dialog box, accessed through the **Promote Properties** property of the **Schema** node, to delete all but one of the Distinguished Field promotions that are associated with the same node in this schema.
