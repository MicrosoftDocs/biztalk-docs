# There are no batch elements to send for party
## Details  
  
| Field | Error Details |
|-----------------|----------------------------------------------------------------------------------------|
|  Product Name   |   [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]   |
| Product Version |               [!INCLUDE[btsEDIVersion](../includes/btsediversion-md.md)]               |
|    Event ID     |                                           -                                            |
|  Event Source   | [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] EDI |
|    Component    |                                    Batching Engine                                     |
|  Symbolic Name  |                                EmptyMessageNotPossible                                 |
|  Message Text   |                   There are no batch elements to send for party {0}                    |
  
## Explanation  
 This Warning indicates that no batch message was sent when the batch trigger was executed because no batch elements had been received.  
  
## User Action  
 No action is necessary.
