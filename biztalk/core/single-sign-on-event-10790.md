# Single Sign-On: Event 10790
## Details  
  
| Field | Error Details |
|-----------------|------------------------------------------------------------|
|  Product Name   |                 Enterprise Single Sign-On                  |

|  Symbolic Name  |                  ENTSSO_E_TICKET_EXPIRED                   |
|  Message Text   |                 The ticket has timed out.                  |
  
## Explanation
