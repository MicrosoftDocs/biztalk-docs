# Error - Functoid Has No Output
**Error Code**  
  
 btm1013  
  
 **Explanation**  
  
 There is no output link for the indicated functoid, rendering it useless.  
  
 **User Action**  
  
 Create an appropriate output link for the indicated functoid.
