| Field | Error Details |
|-----------------|-------------------------------------------------------------------------------------------------------------------|
|  Product Name   |                                             Enterprise Single Sign-On                                             |

|  Event Source   |                                                      ENTSSO                                                       |
|    Component    |                                                        N\A                                                        |
|  Symbolic Name  |                                            SSO_INFO_GOT_CURRENT_SECRET                                            |
|  Message Text   | Got the current secret from the master secret server.%r<br /><br /> Secret Server Name: %1%r<br /><br /> MSID: %2 |
