# Error - Exception Caught
**Error Code**  
  
 btm1023  
  
 **Explanation**  
  
 The identified exception was encountered, indicating a fatal error during map validation or testing.  
  
 **User Action**  
  
 Take corrective action based on the nature of the identified exception.
