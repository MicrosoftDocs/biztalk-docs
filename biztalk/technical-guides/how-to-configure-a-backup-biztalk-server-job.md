You must configure the Backup BizTalk Server job before you can back up BizTalk Server. For more information about how to configure a Backup BizTalk Server job, see [How to Configure the Backup BizTalk Server Job](../core/how-to-configure-the-backup-biztalk-server-job.md) (<https://go.microsoft.com/fwlink/?LinkID=153813>) in [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] Help.
