# HL7 2.2 Folders and Events
The following table lists the subfolders created by the setup wizard within the HL7 version 2.2 folder for HL7-encoded messages. These subfolders contain the schemas used by Microsoft BizTalk Accelerator for HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) to validate, parse, and serialize the events listed in the Events column of this table. The subfolder names describe the types of events these schemas support.  
  
|Subfolder|Events|  
|---------------|------------|  
|Admit, Discharge and Transfer|A01-A37|  
|Master Files|M01-M03|  
|Network Management|N01-N02|  
|Order Entry|O01-O02|  
|Patient Accounting|P01-P04|  
|Query|Q01-Q03,Q05|  
|Observation Reporting|R01-R04|
