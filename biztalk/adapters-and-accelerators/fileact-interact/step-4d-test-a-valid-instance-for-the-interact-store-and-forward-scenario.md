# Step 4D: Test a Valid Instance for the InterAct Store and Forward Scenario
Before you begin this step, you must complete [Step 4C: Create a Test Instance for the InterAct Store and Forward Scenario](../../adapters-and-accelerators/fileact-interact/step-4c-create-a-test-instance-for-the-interact-store-and-forward-scenario.md).  
  
### To test a valid instance  
  
1.  Drop the file ExchangeReqSimple.xml into c:\SWIFTAdapterTutorial\Interact\InputRequest_SnF  
  
2.  Verify that after a moment, the ExchangeReqSimple.xml file disappears from the folder.  
  
3.  Browse to C:\SWIFTAdapterTutorial\Interact\HandleRequest. Verify that Sw:HandleRequest is in the folder.  
  
4.  Open the Sw:HandleRequest message in a text editor, such as Notepad, and verify that the payload section is the same as in the file ExchangeReqSimple.xml.
