# SStatus.bstrErrorMessage Field
A string that contains an error message.  
  
## Syntax  
  
```cpp#  
  
public: BSTR bstrErrorMessage;  
```  
  
## Remarks  
 **bstrErrorMessage** Can be NULL. You must free **bstrEerrorMessage** after every use.
