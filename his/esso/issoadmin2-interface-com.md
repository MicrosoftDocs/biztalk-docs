# ISSOAdmin2 Interface (COM)
The **ISSOAdmin2** interface provides additional administration functions for the Enterprise Single Sign-On (SSO) server database.  
  
## Requirements  
 **Type Library**: SSOAdmin 1.0 Type Library (SSOAdmin.dll)
