# SPasswordChange.ullTimeStamp Field
An integer containing a UTC timestamp in FILETIME format.  
  
## Syntax  
  
```cpp#  
  
public: ULONGLONG ullTimeStamp;  
```  
  
## Remarks  
 If zero, then Enterprise Single Sign-On (ENTSSO) uses the current time.
