# CMDSemSet
The **CMDSemSet** function sets a RAM semaphore.  
  
## Syntax  
  
```  
  
USHORT FAR CMDSemSet(   
ULONG FAR *ramSem   
);  
```  
  
#### Parameters  
 *ramSem*  
 Address of the semaphore.
