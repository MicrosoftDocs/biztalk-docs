# CPI-C Symbolic Name: Security Settings
Specify a User ID and Password  
  
 **User ID**  
 The **User ID** can contain from 1 through 10 characters.  
  
 **Password**  
 The **Password** can contain from 1 through 10 characters.  
  
 **Verify Password**  
 Type the **password** again.
