# Supported RPG Keywords
Transaction Integrator (TI) supports the following Report Program Generator (RPG) keywords:  
  
- CONST  
  
- DATFMT  
  
- DIM  
  
- LIKE  
  
- LIKEDS  
  
- OCCURS  
  
- PACKEVEN  
  
- TIMFMT  
  
  TI does not support the following RPG keywords:  
  
- ALIGN  
  
- ALT  
  
- ALTSEQ(*NONE)  
  
- ASCEND  
  
- BASED  
  
- CTDATA  
  
- DESCEND  
  
- DTAARA  
  
- EXPORT  
  
- EXFLD  
  
- EXTFMT  
  
- EXTNAME  
  
- EXTPGM  
  
- EXTPROC  
  
- FROMFILE  
  
- IMPORT  
  
- INZ  
  
- NOOPT  
  
- OPDESC  
  
- OVERLAY  
  
- PERRCD  
  
- PREFIX  
  
- PROCPTR  
  
- STATIC  
  
- TOFILE  
  
- VALUE
