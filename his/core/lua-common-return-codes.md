# LUA Common Return Codes
This section describes the primary and, if applicable, secondary return codes that are common to the logical unit application (LUA) verbs. The return codes are listed in hexadecimal order.  
  
 Verb-specific return codes are described for the individual verbs in [LUA RUI Verbs](../core/lua-rui-verbs.md) and [LUA SLI Verbs](../core/lua-sli-verbs.md).  
  
 This section contains:  
  
-   [LUA Primary Return Codes](../core/lua-primary-return-codes.md)  
  
-   [LUA Secondary Return Codes](../core/lua-secondary-return-codes.md)
