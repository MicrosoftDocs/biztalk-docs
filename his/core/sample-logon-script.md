# Sample Logon Script
```  
SETTIMEOUT 30,EXIT  
WAITSESSION SSCP  
WAIT 3  
SEND vm1@E  
WAITSESSION LULU  
WAIT 3  
SEND myuser@Tmypass@E  
WAIT 3  
SEND run myapp@E  
WAIT 3  
EXIT:  
```
