# Server Properties: SSO Service
Use this page to enter information for the SSO Service.  
  
## UIElement List  
 **Service Account**  
 Enter the name of the SSO Service Account.  
  
 **Password**  
 Enter a password.  
  
 **Confirm password**  
 Re-enter the password.
