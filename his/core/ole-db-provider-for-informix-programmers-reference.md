# OLE DB Provider for Informix Programmers Reference
The following table summarizes the OLE DB version 2.8 objects that are supported by the current version of MicrosoftÂ® OLE DB Provider for Informix.  
  
|OLE DB object|Support|  
|-------------------|-------------|  
|Command Object|Yes, most interfaces|
